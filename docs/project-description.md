## Project Context
This project is developed as an academic initiative in the field of software engineering.
It aims to simulate the design and development of a real-world web application
addressing a concrete problem faced by citizens in Montreal.

## Target Users
- Students
- Newcomers
- Residents of Montreal

## Main Features (Planned)
- Identification of user situation
- Categorization of essential services
- Guided recommendations
- Simple and accessible user interface

## Technical Approach
The project is designed with a clear separation between frontend and backend.
This approach allows scalability, maintainability, and better understanding
of modern web application architecture.

## Educational Value
This project demonstrates:
- Problem analysis
- System design
- Web development fundamentals
- Documentation and version control
